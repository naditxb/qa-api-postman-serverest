# QA API Postman - ServeRest

Projeto de portfólio QA com testes de API utilizando Postman na API pública ServeRest.

## Objetivo

Validar endpoints principais de usuários e autenticação.

## Ferramentas utilizadas

- Postman
- GitHub
- API ServeRest

## Cenários testados

- Cadastro de usuário com sucesso  
- Busca usuário por ID  
- Login válido  
- Validação de usuário duplicado

## Estrutura do projeto

- postman/ServeRest_Collection.json
- test-cases.md
- bug-reports.md

## Como executar

1. Importar a collection no Postman  
2. Executar requests manualmente  
3. Validar respostas da API
